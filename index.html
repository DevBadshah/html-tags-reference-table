All HTML Tags and Their Uses

Tag Use Example
<html> Root element of an HTML document <html></html>
<head> Contains meta, title, and links <head></head>
<title> Sets the title of the page <title>My Page</title>
<body> Main content of the page <body></body>
<h1> to <h6> Headings (largest to smallest) <h1>Heading</h1>
<p> Paragraph of text <p>This is text.</p>
<br> Line break First Line <br> Second Line
<hr> Horizontal rule <hr>
<strong> Bold text (semantic) <strong>Bold</strong>
<em> Italic text (semantic) <em>Italic</em>
<ul> Unordered list <ul><li>Item</li></ul>
<ol> Ordered list <ol><li>Item</li></ol>
<li> List item <li>Item</li>
<a> Hyperlink <a href="url">Link</a>
<img> Image <img src="image.jpg" alt="text">
<video> Video file <video src="file.mp4"></video>
<audio> Audio file <audio src="file.mp3"></audio>
<table> Table container <table></table>
<tr> Table row <tr></tr>
<th> Table header <th>Header</th>
<td> Table data cell <td>Data</td>
<form> Form container <form></form>
<input> Input field <input type="text">
<button> Button <button>Click</button>
<header> Header section <header></header>
<footer> Footer section <footer></footer>
<section> Section of content <section></section>
<article> Article content <article></article>
